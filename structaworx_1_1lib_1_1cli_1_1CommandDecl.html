Class Description

A declaration for an cli::Command.

Construction is done by passing a custom enum element of an enum type equipped with enum meta data of a distinct type. Such enums are announced to ALib CLI using macro ALIB_CLI_COMMANDS.

When bootstrapping ALib CLI, method CLIApp::DefineCommands has to be invoked for (each) enum type.

The tuple elements have the following meaning:

Index Description
  • - -
0 The underlying integer value of the enum element
1 The name of the command as parsed from command line.
2 The minimum amount of characters to parse to accept the command.
3 List of parameters attached. Separated by /.

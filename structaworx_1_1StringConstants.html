Class Description

template<typename TChar>
struct aworx::StringConstants< TChar >

This empty template class has two specializations for template types nchar and wchar , which each provide static fields:

  • Null,
  • Empty,
  • NewLine and
  • DefaultWhitespaces.

The class is useful to implement generic template methods. In non-generic specializations it is more efficient to use constexpr namespace members

  • NNullString, WNullString, NullString,
  • NEmptyString, WEmptyString, EmptyString,
  • NNewLine, WNewLine, NewLine and
  • NDefaultWhitespaces, WDefaultWhitespaces, DefaultWhitespaces.

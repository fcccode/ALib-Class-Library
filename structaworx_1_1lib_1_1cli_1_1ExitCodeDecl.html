Class Description

An exit code of the cli application.

Construction is done by passing a custom enum element of an enum type equipped with enum meta data of a distinct type. Such enums are announced to ALib CLI using macro ALIB_CLI_EXIT_CODES.

When bootstrapping ALib CLI, method CLIApp::DefineExitCodes has to be invoked for (each) enum type.

The tuple elements have the following meaning:

Index Description
  • - -
0 The underlying integer value of the enum element
1 The name of the exit code (usually same as C++ enum element name.
2 The CLI library exception associated to this exit code.

Announcing the main application's exit codes to the ALib CLI library has two reasons:

  • The exit codes are included in the help output text utility methods provided by class CLIApp.
  • ALib CLI library Exceptions can be translated to valid exit codes using method CLIUtil::GetExitCode.

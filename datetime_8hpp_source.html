30 /** ************************************************************************************************
31  * This class provides an interface into the system's clock values. In contrast to values of sibling
32  * class \alib{time,DateTime}, the underlying system timer is not guaranteed to be steady.
33  * This means that an object created or set at a later point in time of the program execution,
34  * might represent an earlier point in time.
35  * This might happen when the system clock gets adjusted in-between two measurements.
36  *
37  * Therefore, the class is to be used to represent calendar clock values which usually get
38  * converted to human readable formats (calendar dates and 24/60/60 clock times).
39  *
40  * To construct an object of this type using calendrical date an time values, helper class
41  * \b %CalendarDateTime with its constructor
42  * \alib{time,CalendarDateTime::CalendarDateTime(int,int,int,int,int,int,int)} is available.
43  * A typical construction with this helper could look like this:
44  *
45  * DateTime myDate= CalendarDateTime(2018,1,31,14,5).Get( Timezone::UTC );
46  **************************************************************************************************/

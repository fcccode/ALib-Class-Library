Class Description

template<typename T, typename TAllocator, char TSeparator = '/'>
class aworx::lib::util::StringTree< T, TAllocator, TSeparator >::Node

The internal, protected data structure implementing a node of the tree. This class is not accessible from outside. Instead, inner class Cursor, which represents node objects, provides an indirect interface to the values of this class.

Searches and - if no existent - creates a child node.
If a child needs to be created, childName is first checked to not equal to "." or ".." and that it does not contain the separation character. If it does, still true is returned and with nullptr for the new child. Furthermore, an ALIB_WARNING is reported.

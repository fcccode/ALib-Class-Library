Class Description

template<typename TEMD>
struct aworx::lib::cli::ArgumentDecl< TEMD >

Struct used as parent for types

  • CommandDecl,
  • OptionDecl,
  • ParameterDecl and
  • ExitCodeDecl.

Stores runtime information of user-defined enum types.

Construction is done by passing a custom enum element of an enum type equipped with enum meta data of a distinct type.

Such enums are announced to ALib CLI using macros

  • ALIB_CLI_COMMANDS,
  • ALIB_CLI_OPTIONS,
  • ALIB_CLI_PARAMETERS and
  • ALIB_CLI_EXIT_CODES,

When bootstrapping ALib CLI, method CLIApp::DefineExitCodes has to be invoked for (each) enum type.

Class Description

Struct used as parent for types

  • Command,
  • Option and
  • Parameter.

Stores

  • a pointer to the parent application,
  • the position in CLIApp::ArgNOriginal, respectively CLIApp::ArgWOriginal where the object was found and
  • number of arguments consumed when reading the object.
Note
For technical reasons, other members that are shared between the derived types named above, have to be declared (each three times) with the types themselves.

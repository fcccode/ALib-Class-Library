Class Description

This class provides an interface into system dependent steady, monotonic timer values, which are independent from the system's clock settings.

Usually, this class is used to measure and store durations (e.g. with inherited method Age). The class is not suitable to process date and time values. For this, sibling class DateTime is available. However, in some situations it still may be needed to convert points in time created with this class to human readable date and time representations. For this, class TicksConverter allows to convert objects of this type to objects of sibling type DateTime.

As tick values are system dependent, they should not be stored and loaded in data files or otherwise shared between systems. Even on the same machine, the values may become incompatible (e.g. after a reboot).

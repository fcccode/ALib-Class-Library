◆ Read()

bool Read ( ParameterDecl &  decl)

Tries to reads the object from the front of CLIApp::ArgsLeft.

The true on success. If it could not be decided if the actual CLI argument contains this parameter false is returned to indicate that parsing commands has to stop now.

This is done in the following cases:

  • When ParameterDecl::Identifier is empty and the parameter is ParameterDecl::IsOptional gives true.
  • When it was successfully read, but ParameterDecl::QtyExpectedArgsFollowing is defined -1.

See ReadNextCommands for details

41 /** ************************************************************************************************
42  * The class provides static method #FormatParagraph to format a "paragraph"
43  * that starts at a certain index of an \b %AString buffer and reaches to its end.
44  *
45  * When an instance of the class is created, the (public) members of the class provide the text
46  * buffer and the parameters which are needed for the static method. With methods
47  * #Add, member #Formatter is used to append the given parameters and then format the
48  * new paragraph. This way, a longer text might be built by repetitive calls.
49  *
50  * @tparam TChar The character type. Implementations for \c char and \c wchar_t are provided
51  * with type definitions \ref aworx::NSimpleText and \ref aworx::WSimpleText.
52  **************************************************************************************************/

168  /** ****************************************************************************************
169  * Formats one or more paragraphs (separated by \b NewLine symbols) with three optional
170  * features:
171  *
172  * - Wrapping of lines longer than lineWidth (word wrap)
173  * - Justify the text, which here means "full justification", i.e. format the text to have
174  * lines of the exact same width.
175  * - Adding an indentation to each line with an optionally different indentation for
176  * the first line after a \b NewLine symbol and subsequent ones.
177  *
178  * The paragraph starts at \p{startIdx} and all of the rest of the string is treated
179  * as one paragraph. New-line character sequences found within the paragraph are considered
180  * manual line ends. Hence, no block formatting for lines ending with a new line character
181  * is performed.
182  *
183  * The method is static and hence can be used with arbitrary buffers.
184  * Non-static methods #Add invoke this methods after adding the given content
185  * to the internal buffer. Furthermore, convenience functions and corresponding member
186  * variables, simplify the use of this method, when indirectly used through an instance of
187  * the class.
188  *
189  * @param text The text containing the paragraph to format.
190  * @param startIdx The start of the paragraph.
191  * @param lineWidth The width of the line. If \c 0 or negative, no line wrap is
192  * performed.
193  * @param justifyChar If this is unequal to <c>'\0'</c> it denotes the fill
194  * character used to justify the paragraph. If <c>'\0'</c>,
195  * @param[out] maxLineWidth Provides the maximum width of all text lines written.
196  * paragraph is not justified.
197  * @param indentFirstLine The indent string of the first line. Defaults to \c nullptr.
198  * @param indentOtherLines The indent string of subsequent lines. Defaults to \c nullptr.
199  *******************************************************************************************/

234  /** ****************************************************************************************
235  * This method implements a pre-processing of the text prior to #Add paragraphs found in
236  * the text.
237  *
238  * The pre-processing is quite simple. Its purpose is to allow longer strings (e.g. loaded
239  * from resources) with multiple paragraphs to be formatted by embedded escape sequences
240  * to include indents and nested bullet schemes.
241  *
242  * The escape sequences begin with the character stored in field #MarkerChar, which defaults
243  * to <c>'@'</c>. The following table documents the sequences:
244  *
245  * Sequence | Description
246  * ---------|------------------------------------------------------------------------
247  * \@@ | Inserts the marker character itself.
248  * \@>> | Indent text by two spaces
249  * \@<< | Un-indent text by two spaces
250  * \@*> | Increases bullet level.
251  * \@<* | Decreases bullet level
252  * \@P | Inserts a new line (like '\n') but without ending the current and starting a new bullet point.
253  * \@HLc | Inserts a horizontal line of width #LineWidth using \p{c} as fill character.
254  *
255  * The nested bullet point characters are received from vector #MarkerBullets.
256  *
257  * @throws aworx::lib::strings::format::Exceptions
258  * - \alib{strings::format::Exceptions,UnknownMarker}
259  * - \alib{strings::format::Exceptions,EndmarkerWithoutStart}
260  *
261  * @param args The list of arguments to add.
262  ******************************************************************************************/
263  ALIB_API
264  void AddMarked( Boxes& args );

266  /** ****************************************************************************************
267  * Variadic template argument version of #AddMarked.

269  * @throws aworx::lib::strings::format::Exceptions
270  * - \alib{strings::format::Exceptions,UnknownMarker}
271  * - \alib{strings::format::Exceptions,EndmarkerWithoutStart}
272  *
273  * @param args The variadic list of arguments to add.
274  * @return A reference to ourselves to allow concatenated calls.
275  ******************************************************************************************/

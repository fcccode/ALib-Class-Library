34 /** ************************************************************************************************
35  * This class represents a point in time as a set of calendar and clock values
36  * (year, month, day, hour, ...).
37  * It provides methods to convert to and from objects of type \alib{time,DateTime}.
38  * In addition, a method to format the date and time into human readable string value is available.
39  *
40  * \note
41  * The conversion from and into objects of type \alib{time,Ticks} is intentionally not supported.
42  * In the case that such objects should be used with this class, an additional conversion
43  * step has to be performed using class \alib{time,TicksConverter}. In other words, conversion can
44  * be performed as follows:
45  *
46  * CalendarDateTime <=> DateTime <=> Ticks
47  *
48  * \note This class is using system specific calendar methods and relies on the locale and time zone
49  * settings of the machine.
50  **************************************************************************************************/

153  /** ********************************************************************************************
154  * Formats the date using a given pattern string. Within the pattern string, different symbols
155  * are interpreted as tokens. The format is compatible with C# time format strings, as well as
156  * with class SimpleDateFormat of the Java APIs.<br>
157  * Strings within the format text that should not be interpreted as tokens may be surrounded
158  * by single quotes. Two single quotes in a row, will be replaced by one single quote.<br>
159  * The following tokens are supported:
160  *
161  * <center>Token</center> | <center>Description</center>
162  * - - - - -| - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -
163  * y |The year with as many digits as it has (for current dates this is 4).</TD> </TR>
164  * yy |The year, truncated to 2 digits (modulo 100).</TD> </TR>
165  * yyy...y |The year with a minimum amount of digits as amount of y-characters given.</TD> </TR>
166  * M |The month as numbers from 1..12.</TD> </TR>
167  * MM |The month as numbers from 01..12.</TD> </TR>
168  * MMM |The month as abbreviated, 3-digit word defined by resourced strings (defaults to English language).</TD> </TR>
169  * MMMM |The month as word defined by resourced strings (defaults to English language).</TD> </TR>
170  * d |The day as numbers from 1..31.</TD> </TR>
171  * dd |The day as numbers from 01..31.</TD> </TR>
172  * ddd |The day as abbreviated, 3-digit word defined by resourced strings (defaults to English language).</TD> </TR>
173  * dddd |The day as word defined by resourced strings (defaults to English language).</TD> </TR>
174  * H |The hour as numbers from 0..23.</TD> </TR>
175  * HH |The hour as numbers from 00..23.</TD> </TR>
176  * K |The hour as numbers from 0..11 am/pm.</TD> </TR>
177  * KK |The hour as numbers from 00..11 am/pm.</TD> </TR>
178  * m |The minute as numbers from 0..59.</TD> </TR>
179  * mm |The minute as numbers from 00..59.</TD> </TR>
180  * s |The second as numbers from 0..59.</TD> </TR>
181  * ss |The second as numbers from 00..59.</TD> </TR>
182  *
183  * @param format The format pattern string.
184  * @param target A reference to an AString that gets the result of the format processing
185  * appended.
186  * @param targetData If \c CurrentData::Keep (the default) the string is appended to \p{target}.
187  * if \c CurrentData::Clear, \p{target} is cleared.
188  * @returns \p{target} (for convenience).
189  **********************************************************************************************/
190  ALIB_API
191  AString& Format( Substring format, AString& target,
192  lib::lang::CurrentData targetData= lib::lang::CurrentData::Keep );
193 };

195 /** ************************************************************************************************
196  * This class represents a time span, measured in human units like days, hours, minutes and so on.

199  * \alib{time,TimePointBase::Duration,Ticks::Duration} is supported.
200  **************************************************************************************************/

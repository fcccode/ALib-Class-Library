Programmer's Manual

Attention
This is a non-existing namespace! It is exclusively defined for the documentation parser.
In this "documentation namespace", you will find specializations of templated namespace functions and types of namespace aworx::lib::strings. This is of-course wrong in respect to the C++ language definition, which requires such specializations to be placed in the original namespace.
It was decided to 'trick' the documentation parser to show these specialization in this namespace, to keep namespace aworx::lib::strings clean and to have all specializations in one place.
In other words: All types and functions described in this namespace are residing in namespace aworx::lib::strings.

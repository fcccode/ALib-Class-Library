Class Description

template<typename TChar>
class aworx::lib::strings::thirdparty::std::ISReadLineBase< TChar >

Parameter class used for application to objects of type AString with specialization of template struct T_Apply<ISReadLine,TChar>.
Reads a line of text from a std::istream and appends it to the target AString.
While, of-course, this class can be created 'inline' (similar objects of parameter classes found in Format), if a series of lines are to be read from a std::istream, a local object of this type might be created outside the loop and used inside. In this case, the 'output parameter' IsEOF, can be used e.g. as a loop criteria.

See also
  • operator>>(istream&, AString&) and operator<<(ostream&, String&).
  • For a sample, refer to source code of ALib class IniFile, method IniFile::ReadFile.

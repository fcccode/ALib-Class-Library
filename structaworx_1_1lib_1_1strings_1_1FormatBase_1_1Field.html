Class Description

template<typename TChar>
struct aworx::lib::strings::FormatBase< TChar >::Field

Used to create temporary objects which are applied to AString instances.

Appends the given object to the AString using a defined 'field'-width. If the contents of the field is shorter than parameter width specifies, the field is filled with a corresponding amount of padChar characters.
Parameter alignment of type Alignment allows to left-, right- or center-align the contents of the field.

Note
In case, module ALib Boxing is not available, the field content parameter will be of type const String&.
Otherwise, boxing interface IApply will be invoked on the given box internally to receive the string representation.
Therefore, it is mandatory, that for any type that is used with this class to be formatted in a field, this boxing interface has to be implemented. As documented with that interface, for types that are applicable to AString objects already, all that is needed is to use macro ALIB_BOXING_DEFINE_IAPPLY_FOR_APPLICABLE_TYPE with the type in the bootstrap section of an application.

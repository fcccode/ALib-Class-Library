void loadStandardEnumTable ( int  tupleType,
void *  table,
aworx::Library &  library,
const aworx::String &  name

Helper function used with ALib enum meta data when such is defined using either macro ALIB_LANG_ENUM_NAMED or ALIB_LANG_ENUM_PARSABLE, respectively if the meta data types match the ones given with these macros.

The function is invoked by the default implementation of EnumMetaData::CheckLoad if T_Resourced is specialized for the enumeration type.

The trick here here is to avoid template inlining and using a void* pointer for the type of parameter table. In the implementation of this function, the pointer is reinterpreted to the correct type.

Note
While this might not be considered a correct C++ code style, this approach enables ALib Enum Meta Information to be included at a much earlier stage of compilation as it avoids a circular dependency with ALib features for parsing string values.

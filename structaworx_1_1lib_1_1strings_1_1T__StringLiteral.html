Class Description

template<typename TLiteral, typename TChar>
struct aworx::lib::strings::T_StringLiteral< TLiteral, TChar >

This TMP struct is similar to aworx::lib::strings::T_String. It is to be specialized for custom string types that allow to infer the string length already from the TLiteral itself.

The most obvious type are C++ string literals, hence character arrays char[TCapacity] of a given length.

See also class StringLiteral which comes with a (templated) specialization of this struct and can be used to declare C++ strings of fixed size with variable contents.

The benefit of specializing this struct when possible (in comparison to specializing aworx::lib::strings::T_String) is that method Length is a constexpr with this struct. This allows shorter template code and a small performance gain with some methods.

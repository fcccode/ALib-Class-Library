Class Description

template<typename T, int TBlockSize>
struct aworx::lib::util::StringTreeBlockAllocator< T, TBlockSize >

Implementation of StringTreeDefaultAllocator which uses a MemoryBlocks object to allocate the keys.

This helper can be used in cases when a string tree only grows over time and now branches are replaced.

Template Parameters
TThe value type of elements stored in the nodes of the tree.
TBlockSizeThe size of the memory chunks to allocate for key strings.

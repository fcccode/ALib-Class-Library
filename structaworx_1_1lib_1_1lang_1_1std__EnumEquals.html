std_EnumEquals Struct Reference

#include <enum.hpp>

Class Description

Implements a comparison functor for objects of class Enum, usable with types found in namespace std. Instead of implementing the operator inside namespace std, this struct can be provided as template parameter, e.g. to std::unordered_map., for which a templated type definition is provided for with aworx::UnorderedEnumMap.

aworx::lib::strings::thirdparty::std Namespace Reference

Programmer's Manual

With the inclusion of header file alib/compatibility/std_string.hpp, some compatibility between ALib Strings and string types of namespace std is provided.

1. Specializations of T_String:
Specializations of aworx::lib::strings::T_String is given for type std::string. This allows to construct objects of class String directly and implicitly from objects and pointers to objects of std::string.

2. Applicable Types:
Support for applying (appending contents of objects of the types to objects of class AString) is given. Applicable types are:

  • std::string
  • std::wstring
  • std::u16string
  • std::u32string

3. Reading from std::istream:
Class ISReadLineBase is an applicable type used to read lines of textual files from a std::istream object into* an object of class AString.

Attention
The functions operator<< and operator>> as well as the template specializations, in reality are located in namespace aworx::lib::strings For the documentation parser they were moved here to keep the original namespace documentation clean.

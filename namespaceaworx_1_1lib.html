enum Variables
strong

Configuration variables of ALib.

Note
The variables are declared using enum meta data and macro ALIB_CONFIG_VARIABLES. Hence, all variable categories, names, default values and such can be modified by software using ALib by modifying the resource data of the ALib library singleton.

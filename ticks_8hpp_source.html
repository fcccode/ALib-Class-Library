31 /** ************************************************************************************************
32  * This class provides an interface into system dependent steady, monotonic timer values, which are
33  * independent from the system's clock settings.
34  *
35  * Usually, this class is used to measure and store durations (e.g. with inherited method
36  * \alib{time::TimePointBase,Age}). The class is \e not suitable to process date and time values.
37  * For this, sibling class \alib{time,DateTime} is available. However, in some situations it still
38  * may be needed to convert points in time created with this class to human readable date and
39  * time representations. For this, class \alib{time,TicksConverter} allows to convert objects of
40  * this type to objects of sibling type \b %DateTime.
41  *
42  * As tick values are system dependent, they should not be stored and loaded in data files or
43  * otherwise shared between systems. Even on the same machine, the values may become
44  * incompatible (e.g. after a reboot).
45  **************************************************************************************************/
